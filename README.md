# Bellas Boutique - Frontend Profesional

Proyecto base en React + Vite + Tailwind CSS para una tienda online con:

- Login y registro
- Catálogo de productos
- Carrito de compras
- Checkout y factura
- Soporte al cliente
- Panel de administración

## Instalación

```bash
npm install
npm run dev
```

## Estructura

- `src/components`: componentes reutilizables
- `src/pages`: páginas del sistema
- `src/context`: estado global
- `src/data`: datos simulados

## Recomendación para práctoca de otros desarrolladores

Cambien:

- colores
- textos
- nombre visual de algunas secciones
- imágenes
- distribución pequeña de componentes

Así lo usarán como base profesional mientras aprenden los conceptos.
